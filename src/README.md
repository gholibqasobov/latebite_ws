# LateBite — демо-сайт для презентации инвесторам

Прототип платформы по спасению еды (в стиле Too Good To Go) для Алматы.
**SAVE FOOD · SAVE MONEY · SAVE THE PLANET.**

Это кликабельное демо: интерфейс выглядит и работает «как настоящий», но
платежи Kaspi, вход и уведомления — имитация (см. раздел «Что настоящее, а что имитация»).

---

## Запуск

Нужен Node.js 18+ (в этом окружении установлен в `~/.local/node`).

```bash
npm install      # установить зависимости (один раз)
npm run dev      # запустить дев-сервер → http://localhost:5173
npm run build    # собрать продакшн-сборку в dist/
npm run preview  # посмотреть собранную версию
```

## Деплой (бесплатно)

Подойдёт любой статический хостинг. Например, Vercel или Netlify:
загрузите репозиторий, команда сборки `npm run build`, папка вывода `dist`.

## Что есть в демо

| Раздел | Маршрут | Описание |
|---|---|---|
| Главная | `/` | Слоган, «Почему LateBite», «Как это работает», эко-трекер со счётчиками |
| Предложения | `/offers` | Интерактивная карта (Leaflet/OSM) + лента блюд + фильтры |
| Корзина | `/cart` | Блюда из разных заведений, количество, итог |
| Оплата | `/checkout` | Имитация оплаты Kaspi → QR-код заказа |
| Личный кабинет | `/account` | Активные заказы (QR), история, эко-трекер, рефералы, профиль |
| Кабинет партнёра | `/partner` | Меню (CRUD), заказы, QR-сканер, аналитика, отзывы, профиль |
| О проекте | `/about` | Миссия, ЦУР 12 и 13, партнёры, отчёты |
| Контакты | `/contacts` | Форма обратной связи, соцсети |
| Вход | `/login` | Демо-вход (имитация) |

## Что настоящее, а что имитация

**Настоящее:** карта, фильтры, корзина, генерация QR-кодов, графики аналитики,
адаптивная вёрстка, анимированные счётчики, CRUD меню партнёра.

**Имитация (для запуска потребуется бэкенд/договор):**
оплата Kaspi.kz, авторизация, push/email-уведомления, синхронизация заказов
между клиентом и партнёром в реальном времени.

## Структура

```
src/
  components/   переиспользуемые компоненты (Navbar, DishCard, MapView, ...)
  context/      состояние корзины и тостов
  data/         ВСЕ демо-данные и ссылки на изображения  ← правьте здесь
  lib/          форматирование (₸, склонения)
  pages/        страницы по маршрутам
public/         статика (favicon)
MATERIALS.txt   что нужно прислать (фото, лого) и как заменить заглушки
```

## Замена контента и картинок

- **Изображения:** все ссылки в одном файле — `src/data/images.js`. Подробно — в `MATERIALS.txt`.
- **Блюда, цены, заведения:** `src/data/dishes.js`, `src/data/restaurants.js`.
- **Контакты/соцсети:** `src/components/Footer.jsx`, `src/pages/Contacts.jsx`.
- **Цвета бренда:** `tailwind.config.js` (primary `#1A4D3E`, mint `#A8E6CF`, accent `#F4D03F`).

> ⚠️ Названия/логотипы реальных брендов (PAUL, Costa и т.д.) защищены авторским
> правом. Для публичного показа см. рекомендации в `MATERIALS.txt`.
